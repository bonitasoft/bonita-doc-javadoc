Path: / API / managementAPI / setDefaultPolicy / {newPolicy} / {negateRules}

Update the policy for RuleType.PROCESS_START

Path parameters:
negateRules - if set to true all the rules will be updated to respect the new policy. If set to false the rules will not be updated.
newPolicy - the {@link PrivilegePolicy} to apply PrivilegePolicy.ALLOW_BY_DEFAULT: Means that by default all entities are allowed to act on everything. Except the items listed in rules.
PrivilegePolicy.DENY_BY_DEFAULT: Means that by default all entities are NOT allowed to act on anything. Except the items explicitly listed in rules.
