Path: / API / runtimeAPI / skip / {activityInstanceUUID}

Skips the activity if the activity has READY or FAILED state. The execution jumps to the next activity
If successful, this operation changes task state from READY or FAILED to SKIPPED.

Path parameters:
activityInstanceUUID - the activity instance UUID.
