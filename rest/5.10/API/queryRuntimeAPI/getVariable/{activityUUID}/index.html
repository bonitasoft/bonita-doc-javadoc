Path: / API / queryRuntimeAPI / getVariable / {activityUUID}

Obtains a variable for the given activity and variable name. This variable could be local to the activity or global to the process. The activity should either be executed or currently pointed by the a process execution.

For XML Type:

  • getVariable(activityUUID, "myXmlData") returns a Document
  • getVariable(activityUUID, "myXmlData" + BonitaConstants.XPATH_VAR_SEPARATOR + "/root/node") returns a Node
  • getVariable(activityUUID, "myXmlData" + BonitaConstants.XPATH_VAR_SEPARATOR + "/root/@attribute") returns a String
  • getVariable(activityUUID, "myXmlData" + BonitaConstants.XPATH_VAR_SEPARATOR + "/root/text()") returns a String
  • getVariable(activityUUID, "myXmlData" + BonitaConstants.XPATH_VAR_SEPARATOR + anyOtherKindOfXPathExpression) returns a Node

Path parameters:
activityUUID - the activity UUID.
