Path: / API / repairAPI / instantiateProcessWithInstanceIniciator / {processDefinitionUUID}

Starts a new process instance, positioned at specific activities, with a map of instance's variable values. The activities located before the start activities are not executed. All connectors of specified activities will be executed normally (include connectors OnReady).

Path parameters:
processDefinitionUUID - the process definition UUID
