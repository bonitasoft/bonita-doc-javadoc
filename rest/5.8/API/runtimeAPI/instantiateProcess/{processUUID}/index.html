Path: / API / runtimeAPI / instantiateProcess / {processUUID}

Creates an instance of the specified process and start the execution at the specified activity. Specified activity must be a start activity (no incoming transitions). returned instance has STARTED state. If the first activity has StartMode=manual then a task has been created. If the first activity has StartMode=automatic then the automatic behavior of the activity has been started.
