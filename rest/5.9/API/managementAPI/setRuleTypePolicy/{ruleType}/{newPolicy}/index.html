Path: / API / managementAPI / setRuleTypePolicy / {ruleType} / {newPolicy}

Update a rule type policy.

Path parameters:
ruleType - the rule type
newPolicy - the {@link PrivilegePolicy} to apply to the rule type PrivilegePolicy.ALLOW_BY_DEFAULT: Means that by default all entities are allowed to act on everything. Except the items listed in rules.
PrivilegePolicy.DENY_BY_DEFAULT: Means that by default all entities are NOT allowed to act on anything. Except the items explicitly listed in rules.
